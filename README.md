# Intelligent Multimodal Travel Planning Agent

## Overview
An intelligent mobile agent that plans a trip by navigating real Android apps such as Google Maps and Booking.com.

## What It Does
- Discovers popular destinations using Google Maps
- Finds the cheapest stay using Booking.com
- Adapts to dynamic mobile UI flows
- Provides a safe, recommendation-based travel plan

## What It Does NOT Do
- No automatic payments
- No fragile UI scraping
- No unsafe booking actions

## Tech Stack
- Python
- adbutils (ADB-based Android automation)
- Real Android device

## Agentic Behavior
The agent adapts its behavior based on app state and UI flow, prioritizing robustness and cost optimization.
